Authorization Steps

  1. Click the "Authorize Gmail Access" button below
  2. You'll be redirected to Google's authorization page
  3. Grant permission to access your Gmail account
  4. You'll be redirected back to this application
  5. Once authorized, you can sync Gmail emails

Actions

Authorize Gmail Access

API Endpoints

GET /api/v1/login/oauth2/authorize/google - Start OAuth2 flow
GET /api/v1/login/oauth2/status - Check authorization status
POST /api/v1/login/oauth2/revoke - Revoke authorization
POST /api/v1/gmail/sync - Sync Gmail emails (requires authorization)
GET /api/v1/gmail/unsynced-count - Get unsynced emails count
